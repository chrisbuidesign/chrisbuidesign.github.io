Dotted

Role

Interaction Designer, Developer

Timeline

2 weeks

Format

Web

Context

I designed and developed a data visualization from a large data set.

One of my class assignments challenged me to create a visualization using images that my classmates had collected over the course of two weeks.

Goals & Challenges

Create a compelling data experience by including some form of interaction.

Initial Sketches (draw some more)

Development

I used Processing to import images from Flickr, and to find the average of their R, G, and B values. I took the average and graphed it as a point on the graph. Their R, G, and B values corresponsing to the x, y and z coordinates, respectively.

What's Next?

If I were to return to this project, the first thing I would do is add a function that allows users to see individual images. It would be interesting to hover over each dot and see what was behind the color. I would also add a panning functionality so that users can see the visualization from an orthogonal view.
